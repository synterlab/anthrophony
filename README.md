# ANTHROPHONY
### *The Sound of Humanity*

> A playable archive of human cultural memory.

---

## What is ANTHROPHONY?

ANTHROPHONY is an interactive cultural exploration game where users investigate humanity through sound. Users become explorers, researchers, archivists, and curators. Music is only the entry point — the real subject is human civilization.

## Features

- **Discover** — Identify acoustic traditions from field recordings via multiple-choice challenges
- **Explore** — Interactive rotating globe with 9 world culture regions, each with detailed historical context
- **Collection** — Personal museum of 8 rare artifacts with Sound DNA similarity visualization
- **Expeditions** — Story-based timeline missions across human history
- **Profile** — Academic progression system with ranks and achievements
- **The Archivist** — AI companion panel for cultural inquiry

## Design

- Mobile-first, cinematic dark UI inspired by National Geographic, Duolingo, and Civilization
- Pure HTML/CSS/Vanilla JS — zero dependencies, zero build step
- Works immediately when opened

## Stack

```
Single file: index.html
No frameworks. No build tools. No dependencies.
Pure HTML + CSS + Vanilla JavaScript.
```

## Deploy

**Vercel (recommended):**
1. Connect this repository to Vercel
2. Set Output Directory to `.` (root)
3. Deploy — live in seconds

**GitHub Pages:**
1. Go to Settings → Pages
2. Set source to `main` branch, `/ (root)` folder
3. Your site will be live at `https://synterlab.github.io/anthrophony`
